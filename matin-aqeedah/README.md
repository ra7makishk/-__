# ملازم مذاكرة العقيدة – برنامج متين المستوى الثاني

ملازم منظمة لمذاكرة محاضرات كتاب التوحيد للشيخ محمد بن عبد الوهاب رحمه الله.

## الملفات

| الملف | المحتوى |
|-------|---------|
| `index.html` | الصفحة الرئيسية |
| `lqa2.html` | ملزمة اللقاء الثاني – ١ محرم ١٤٤٨ |
| `lqa3.html` | ملزمة اللقاء الثالث – ٨ محرم ١٤٤٨ |

## نشر على GitHub Pages

1. ارفعي الملفات على GitHub
2. اذهبي لـ Settings → Pages
3. اختاري Source: `main` branch → `/root`
4. سيكون الموقع على: `https://username.github.io/repo-name/`
